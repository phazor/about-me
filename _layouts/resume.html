Projects

Virtual Cloud Chamber • Oct 2016

Skills learned: HTML, Javascript, Canvas.

I was inspired after seeing a live Cloud Chamber (a device which creates vapor trails when particles pass through) at CERN. I decided to create a true-to-physics simulation of this cloud chamber to show how weird reality can be. It's available at phazor.github.io/cloud-chamber.

Maximum Subarray Problem • Nov 2011 — Feb, 2012

Skills learned: CUDA, C++, Parallel Programming, Algorithms, Image Processing.

My goal for this 3 month contract was to implement various improvements on an image processing algorithm being developed at the University of Canterbury by a theoretician, and measure the efficiency. These algorithms are designed for use on parallel processors so I wrote code for IBM Blue Gene as well as for nVidia cards using CUDA.

Striking a balance between theory and implementation was a nice challenge and appealed to my inner engineer.

Visual Processing for the Bionic Eye • Nov 2009 — Feb, 2010

Skills learned: Python, Population Modelling, Visual Processing.

At National ICT Australia I wrote a simulation of the retina and visual cortex's electrical properties, to help researchers design better bionic eyes. I enjoyed combining my physics and programming skills for something that could benefit society.

Various other projects

A collection of small personal projects. Tools used: React, React-Router, Canvas, CSS, Webkit, Jekyll, available at phazor.github.io.
